# 文档中的访问权限要求

Adobe Workfront Fusion文档中的操作说明文章包含一个表格，其中说明了该过程所需的访问和权限要求。 通过此“访问要求”表格，您可以了解是否可以在Fusion中执行某个操作，或者无法执行某个操作的原因。 本文介绍访问要求表的每个元素，并提供故障排除提示和指向更深入信息的链接。

如果在给定项目的“访问要求”表中没有一行，则对该操作没有该类型的要求。

某些行包含标记为“新”和“当前”的信息。 这是因为Workfront正在过渡到新的定价和打包模型，一些组织在新模型下运营，而其他组织仍在使用当前模型。 要了解您的组织使用哪种模型，请联系您的Workfront管理员。 您可以在本文的[访问要求表](#the-access-requirements-table)部分中查找详细信息以及信息链接。

>[!NOTE]
>
>如果您对此表中的任何字段如何应用有疑问，请与Workfront Fusion管理员联系。
