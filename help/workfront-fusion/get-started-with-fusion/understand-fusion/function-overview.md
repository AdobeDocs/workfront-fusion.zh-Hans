# 函数概述

映射项时，可以使用函数创建简单或复杂的公式。 可用的函数类似于Excel中的函数以及某些编程语言中的函数：

* 它们评估一般逻辑、数学、文本、日期和数组。
* 它们允许您执行条件逻辑和项值的转换，例如将文本转换为大写、修剪文本、将日期转换为不同格式等。
